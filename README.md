# UD25-API04

## URL de la API

https://ud2504.herokuapp.com/

### Endpoints de SALAS

  1- Obtener una lista de los salas
  - GET /api/salas
  
  2- Obtener una sala por su codigo
  - GET /api/salas/{codigo}
  
  3- Insertar una sala nuevo desde un request body
  - POST /api/salas
  
  4- Actualizar una sala indicando su codigo y sus campos en el body
  - PUT /api/salas/{codigo}
  
  5- Borrar una sala por su ID 
  - DELETE /api/salas/{codigo}
